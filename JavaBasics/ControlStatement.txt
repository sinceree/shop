选择结构

if单选结构：
if (表达式){
	语句块;
}

if-else双选择结构：
if (表达式){
	语句块1;
}else{
	语句块2;
}

if-else if-else多选择结构：
if (表达式){
	语句块1;
}else if{
	语句块2;
}else{
	语句块3;
}

switch多选择结构：
switch (表达式){
	case 值1;
		break;
	case值2;
		break;
	default:
		case值3;
}


循环结构

while循环结构
while(循环条件){
	循环体;
}
注：先判断后执行

do-while循环结构
do{
	循环体;
}while(循环条件);
注：先执行一次，后进行判断

for循环结构
for(表达式;循环条件;迭代因子){
	循环体;
}

break语句：用于直接结束循环体，不执行循环中剩余的语句
continue语句：用于跳过某次循环，并继续执行剩余循环语句

语句块：语句块中声明的变量仅作用于语句块内，外部不可使用语句块内定义的变量。
{
	语句块
}

方法：
修饰符 返回值类型 方法名(形参){
	语句;
}
形式参数：方法声明时用于接收外界传入的数据
实参：调用方法时实际传给对方的数据
返回值：再执行完毕后返还给调用它的环境的数据
返回值类型：事先约定的返回值类型
方法调用方式：对象名.方法名(实参);
注：
实参的类型、顺序、数目，需要跟形参一一对应
传递参数传递的是数据的copy值
ruturn终止方法的运行，并指定返回的数据

方法的重载：
方法名称相同，但对于形参的类型、个数、顺序都是不同的。
